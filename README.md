# Holon
Holon keeps your username and avatar synchronized across multiple platforms, and alert when they differ.

> Everything bellow was generated by Claude and I'll use as a Guide to help me build this.

--- 

## Overview

Monitor and sync your profile information across different websites. Get alerts when usernames or avatars don't match across your accounts.

## Supported Platforms

### Tier 1 - Easy (Start Here)
- [YouTube](https://developers.google.com/youtube/v3)
- [GitHub](https://docs.github.com/en/rest)
- [Discord](https://discord.com/developers/docs)
- [Medium](https://github.com/Medium/medium-api-docs)
- [Chess.com](https://www.chess.com/news/view/published-data-api)
- [Lichess.org](https://lichess.org/api)
- [CodeForces](https://codeforces.com/apiHelp)

### Tier 2 - Medium
- [Gmail](https://developers.google.com/gmail/api)
- [Slack](https://api.slack.com/)
- [Reddit](https://www.reddit.com/dev/api)
- [LeetCode](https://leetcode.com/graphql) (GraphQL, undocumented)
- [Telegram](https://core.telegram.org/api)
- [Mastodon](https://docs.joinmastodon.org/api/)
- [Bluesky](https://docs.bsky.app/)

### Tier 3 - Hard (Facebook/Meta)
- [Instagram](https://developers.facebook.com/docs/instagram-api) (Meta Graph API)
- [Facebook](https://developers.facebook.com/docs/facebook-login)
- [Threads](https://developers.facebook.com/docs/threads) (Meta Graph API)
- [LinkedIn](https://www.linkedin.com/developers)
- [WhatsApp](https://www.whatsapp.com/business/api) (Business API only)

### Tier 4 - Very Hard / Unavailable
- Gmail (limited public data)
- Quora (no official API)
- Clubhouse (no public API)
- StackExchange (limited data available)
- TikTok (restricted API)
- ICPC (no public API)
- X/Twitter (paid API - [docs](https://developer.x.com/en/docs/x-api))

## Configuration

1. Create API credentials for each platform
2. Store in `config/credentials.json` (don't commit this)
3. Run the sync tool
